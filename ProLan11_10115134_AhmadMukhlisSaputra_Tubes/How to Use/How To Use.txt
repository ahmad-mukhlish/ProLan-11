Program Restoran Sederhana berbasis GUI dan SQL online

Untuk menggunakan program, pastikan komputer Anda terkoneksi ke internet.

Cara Running program ada 2 :

1. Via Program Restoran Sederhana GUI SQl online.jar
2. Run program pada cmd dengan sintak :

java -cp .;"direktori_mysql_connector";"direktori_classes" main.Main

pada komputer penulis cara running program adalah :
